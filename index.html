A franco le gusta la grandes

map():Transforma cada elemento del array y devuelve un nuevo array.

filter():Crea un nuevo array con todos los elementos que cumplen una condición.

reduce():Ejecuta una función sobre un acumulador y cada elemento, reduciéndolo a un único valor.

find():Devuelve el primer elemento que satisface la condición de prueba.

sort():Ordena los elementos del array en su lugar (modifica el original).

includes():Comprueba si un array contiene un determinado elemento, devuelve true o false.

indexOf():Devuelve el índice de la primera aparición de un elemento.

some():Comprueba si al menos un elemento cumple la condición, devuelve true o false.

every():Comprueba si todos los elementos cumplen la condición, devuelve true o false.

slice():Devuelve una copia superficial de una porción del array en un nuevo array.

splice():Cambia el contenido de un array eliminando, reemplazando o agregando elementos.

split()(Método de String):Divide un string en un array de subcadenas, usando un separador.
